## üîß CUSTOMIZATION GUIDE

### Changing Primary Color
Edit in `theme.css`:
```css
:root {
  --sg-primary: #10b981;        /* Change this */
  --sg-primary-dark: #059669;
  --sg-primary-light: #6ee7b7;
}
```

### Adjusting Animation Speed
Edit in `theme.css`:
```css
:root {
  --transition-fast: 150ms;     /* Decrease for faster */
  --transition-base: 250ms;     /* Default speed */
  --transition-slow: 350ms;     /* Increase for slower */
}
```

### Adding New Animations
Edit `animations.css` and add new `@keyframes` animations.
